### Development Environment

- Xcode 12.2

How to Build
--------------------------

1. Run following commands to resolve dependencies.
    - `pod install`
1. Open `KyotoTrip.xcworkspace` in Xcode.
1. Build "KyotoTrip" scheme in the workspace.
